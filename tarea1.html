📜 Resumen base

CSS (Cascading Style Sheets) es un lenguaje que se utiliza para dar estilo a las páginas web. Permite cambiar colores, tamaños, posiciones y fondos de los elementos HTML. Gracias a CSS se puede separar el diseño visual del contenido de una página web, lo que facilita el mantenimiento y organización del código.
